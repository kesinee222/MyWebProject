# Queuie
โปรเจคระบบค้นหาอู่ซ่อมรถและคาดการณ์การซ่อมบำรุง จัดทำโดย เกศินี ทองลอย

## Features
* ใช้ GPS ในโปรแกรม
* แสดงแผนที่ และพิกัดของผู้ใช้ พร้อมพิกัดอู่ซ่อมรถใกล้เคียง ระยะ 5 กม.
* ดูการคาดการณ์งานซ่อมบำรุง
